# SOP Safety Bot

## Description
**SOP Safety Bot** is a Python script that helps users understand and follow Standard Operating Procedures (SOPs) and safety guidelines. The bot can provide explanations, safety tips, and clarifications based on user input.

## Features
- Interactive Python-based bot
- Explains SOPs in a simple, user-friendly way
- Provides safety tips and recommendations
- Easy to extend for different SOPs or workflows
